# עץ הורים - שלב ב

להסבר על המטלה ראו 
[שלב א](../../../ancestor-tree-a).

בשלב ב עליכם לכתוב מימוש מלא של המחלקה. 
המימוש צריך לעבור את כל הבדיקות - שלכם ושלנו.

אנחנו מריצים את הפקודה

<div dir='ltr'>

	bash grade

</div>

שמריצה את הפקודות:

<div dir='ltr'>

	make test
	./test

</div>

שימו לב: אין לשנות קבצים קיימים אלא רק להוסיף קבצים חדשים.
